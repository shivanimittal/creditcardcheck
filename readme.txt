This is a credit card processing application. 
Below are the features of this application - 

1. Add a new credit card. The credit card will be validated using Luhn 10.
2. Display all the existing card details.

The front end is built using React JS.
The back end is Restful API.

Steps to build
1. Run mvn clean install
2. Run mvn spring-boot:run
3. Open browser and enter the url - http://localhost:8080
